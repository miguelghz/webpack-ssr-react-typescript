# webpack-ssr-react-typescript

Webpack 4 simple boilerplate for Server Side Rendering with Node, React 16 with Typescript 3 & SASS

![Webpack simple](node-ts-react.png)

# Includes
- Node v11
- Express v
- React v16
- Typescript v3
- Sass loader (for client config)

# Instructions

- Clone this repo
- `npm install`
- `npm run build`
- `npm start`
